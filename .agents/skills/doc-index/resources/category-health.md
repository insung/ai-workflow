# 카테고리 건강성 기준 (Category Health Criteria)

INDEX.md의 카테고리 구조를 아래 4개 기준으로 점검하라. 1개 이상 미충족 시 WARN을 반환하라.

## 기준 1: 균형 (Balance)
- 단일 카테고리에 전체 유니크 파일 수의 50%를 초과하여 배치하지 마라. 교차 참조된 파일은 1건으로 카운트하라.
- **위반 시:** 해당 카테고리를 하위 주제 기준으로 분할하라.

## 기준 2: 최소 단위 (Minimum Size)
- 파일이 1개뿐인 카테고리를 허용하지 마라.
- **위반 시:** 가장 유사한 카테고리에 병합하거나, "미분류" 카테고리에 임시 배치하라.

## 기준 3: 응집도 (Coherence)
- 카테고리 내 모든 파일의 1문장 요약이 카테고리명의 주제와 관련되어야 한다.
- **위반 시:** 해당 파일을 더 적합한 카테고리로 이동하라.

## 기준 4: 교차 참조 상한 (Cross-reference Limit)
- 단일 파일이 3개를 초과하는 카테고리에 등재되지 않도록 하라.
- **위반 시:** 가장 관련도가 낮은 카테고리에서 해당 파일을 제거하라.
