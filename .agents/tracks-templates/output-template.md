# Output 템플릿

## 새 트랙 생성 시
```
✅ 트랙 생성: {트랙명}
🎯 목표: {plan.md의 목표 섹션 내용}
📊 Status 갱신: tracks/YYYY-MM-status.md → 미착수 (진행 시, 진행 중으로 변경)
```

## 트랙 완료 시
```
🎉 트랙 완료: {트랙명}
🎯 목표: {plan.md의 목표 섹션 내용}
✅ 완료된 작업: {plan.md 체크박스 총 개수}개
📊 Status 갱신: YYYY-MM-status.md → ✅ 완료
📋 발견 항목: {notes/discovered.md 항목 개수}개 → 별도 트랙 검토 필요
📝 Audit 완료: 미기록 대화 {개수}건 추가 기록
```
